## Api for llm monitoring

### Log Question

```POST /api/log-question
Request Body:
{
  question: string,        
  userId?: string,         
  sessionId?: string,     
  model?: string,          
  timestamp?: string       
}

// Response:
{
  success: boolean,        
  questionId: string       
}```

### Log Answer

```POST /api/log-answer
Request Body:
{
  questionId: string,      
  answer: string,          
  responseTime?: number,   
  model?: string,          
  tokensUsed?: number,     
  error?: string           
}

// Response:
{
  success: boolean,        
  questionId: string       
}```
